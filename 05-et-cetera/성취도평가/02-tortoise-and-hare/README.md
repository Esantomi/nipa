# 토끼와 거북이 경주 결과
### “나랑 달리기 시합하지 않을래?”

토끼와 거북이가 달리기 시합을 하기로 했어요.
공정한 경쟁을 위해서 1초마다 토끼와 거북이의 위치를 다른 동물이 기록하기로 하고 경주를 했네요.

그 위치 데이터가 `csv` 파일로 저장되어 있어요. 우리는 `csv` 파일을 읽어서 토끼와 거북이의 시간별 위치를 그래프로 시각화해 보고자 합니다!
